# Navient (navient)
Navient Corporation is a U.S. corporation based in Wilmington, Delaware, whose operations include servicing and collecting student loans. Following federal loan servicing transitions, education loan servicing was largely transferred to MOHELA. This repository tracks public APIs, developer tools, and machine-readable artifacts for Navient.
